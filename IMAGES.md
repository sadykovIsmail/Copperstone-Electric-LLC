# Image Files — Drop Your Photos Here

Place your image files directly in this `public/` folder with these exact names:

| File name       | Used in section          | Recommended size  | Orientation |
|-----------------|--------------------------|-------------------|-------------|
| `tools.jpg`     | Hero background          | 1920 × 1080 px    | Landscape   |
| `feature.jpg`   | Feature split section    | 1200 × 800 px     | Landscape   |
| `hero.jpg`      | About section            | 900 × 1100 px     | Portrait    |

## Tips
- `tools.jpg` — wide shot, workbench / breaker panel / tools. Dark enough for white text overlay.
- `feature.jpg` — close-up detail shot, e.g. copper wires in a breaker box (like the screenshot).
- `hero.jpg` — professional portrait of electrician on a job site.
- JPG format is fine. PNG also works. Keep file size under 1 MB for fast loading.
- Just replace the file with the same name — the site will update automatically.
